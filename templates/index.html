PROJECTS

Project Details

Smart driving system using android

Technology

Arduino microcontroller and an Android app interface.

Need for this project

Nowadays, there are improving accident cases happening due to phone calls while driving. Sometimes, we are unable to pick up the phone calls while driving. This may go weird during emergency purposes. To avoid such accidents and to overcome such drawbacks of emergency calls, this system can be implemented.

Concept
  • This system uses an android application which automatically changes the phone's profile into 'driving mode' when driving.
  • Under driving mode the following features are available
  • The phone will be switched to 'silent mode'.
  • When a call is received the caller will get a voice msg about the driving condition of the user.
  • Meanewhile if the call is an emergency the caller is provided with the option to connect to the user immediately.
  • Then the user will get an intimation of the call on the display of the vehicle.
  • The user has to off the engine to attend the call, thus ensuring safety to the user.
  • This application activates the driving mode automatically when the user gets to drive. This automation of the mode can be achieved by using Bluetooth pairing as follows:
  • A device (a micro-controller with Bluetooth modem) is set up in the vehicle.
  • When the user's Bluetooth from the mobile gets connected to the vehicle's Bluetooth, the android app changes the phone's profile to driving mode.
  • Further the vehicle's engine can be started only when the Bluetooth connection is paired and can attend or make a call only after the engine is off. This feature enables safety driving.
  • The option for multiple users can be achieved as follows:
  • For default user of the vehicle, Bluetooth gets paired automatically. Only the Bluetooth must be made on for pairing by the user.
  • For other users, a password must be provided for pairing the device for each time, thereby providing security to the vehicle.
  • OTHER ADVANTAGES
  • A tracking device can be set in to the microcontroller device which can be used to track the vehicle in case of theft.
